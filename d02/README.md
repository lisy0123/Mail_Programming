# d02

Fibonacci sequence starts with 0 and 1 where each fibonacci number is a sum of two previous fibonacci numbers. 

Given an integer N, find the sum of all even fibonacci numbers.



ex)

Input: N = 12

Output: 10 

> 0,1,2,3,5,8
>
> even numbers 2 + 8 = 10
