# d03

Given an inter N, find the number of possible balanced parentheses with N opening and closing brackets.



> Input: 1
>
> Ouput: ["()"]

> Input: 2
>
> Ouput: ["(())", "()()"]

> Input: 3
>
> Ouput: ["((()))", "(()())", "()(())", "(())()", "()()()"]
